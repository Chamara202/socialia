HTML (HyperText Markup Language)

HTML (HyperText Markup Language): HTML යනු වෙබ් පිටුවක අන්තර්ගත ව්‍යුහය සැකසීමට භාවිතා කරන markup භාෂාවකි. HTML tags මඟින් headings, paragraphs, lists, tables, forms, සහ media (images, videos, audio) එකතු කිරීමට සහ ව්‍යුහගත කිරීමට හැක.

HTML ලේඛනයක් html tag එකක ඇතුළත් වන අතර, head සහ body යන කොටස් දෙකකින් සෑදී ඇත. head හි meta information (charset, title, links) සහ body හි actual content (text, images) අඩංගු වේ.

HTML5 මඟින් semantic tags (header, footer, article, section) යොදා වෙබ් පිටු Accessibility සහ modern applications සඳහා canvas, SVG, සහ audio/video support සලසයි. a (anchor) tag එක hyperlinks නිර්මාණය සඳහා භාවිතා වේ. table සහ form tags මඟින් tables සහ user input fields නිර්මාණය කළ හැක. HTML attributes (e.g., id, class, src, href) මඟින් customizations කළ හැකි වන අතර HTML සෘජුවම design හෝ interaction සඳහා පාලනයක් සපයන්නේ නැත.

CSS (Cascading Style Sheets)

CSS මඟින් HTML වෙබ් පිටු සැරයන් (styles) සකස් කරයි. CSS styles inline, internal, සහ external ලෙස HTML-ට එකතු කළ හැක. External CSS files tag එක මඟින් HTML-ට සම්බන්ධ කර consistent design ලබා දේ. Selectors (e.g., element selectors, class selectors, ID selectors, pseudo-selectors) මඟින් HTML elements styled කළ හැක. CSS මඟින් media queries භාවිතා කර responsive designs සහ mobile-friendly layouts නිර්මාණය කළ හැක. CSS animations සහ transitions මඟින් user interaction වැඩිදියුණු කරයි. CSS frameworks (e.g., Bootstrap, TailwindCSS) මඟින් predefined classes භාවිතා කර rapid design development කළ හැක. CSS dynamic functionality සඳහා JavaScript මඟින්เสริม විය යුතුය.

JavaScript

JavaScript client-side scripting භාෂාවක් වන අතර, dynamic functionality සහ interactive elements එකතු කිරීමට භාවිතා වේ. Event listeners මඟින් onclick, onmouseover වැනි user interactions handle කළ හැක. Node.js මඟින් JavaScript server-side applications සඳහා භාවිතා කළ හැක. JavaScript browser-dependent වන අතර, large-scale projects සඳහා frameworks අවශ්‍ය වේ.

SQL static nature-ය dynamic behavior handle කිරීමට JavaScript/PHP වැනි languages-සමඟ සමූහ කළ යුතුය.
